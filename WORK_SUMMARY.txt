LESSONS LEARNED
---------------
- Android keyboard handling is notoriously difficult
- adjustResize vs adjustPan each have tradeoffs
- Compose's imePadding() can conflict with windowSoftInputMode
- Scaffold's built-in insets handling can interfere with custom solutions
- Sometimes need to handle insets at multiple levels (Manifest + Compose)
- Box with absolute positioning (align) can keep elements fixed
- weight() modifiers don't always work well with keyboard resizing
