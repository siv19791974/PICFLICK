4. BUG FIXES
------------

A. Coroutine Scope Cancellation (Critical Fix)
Problem: LeftCompositionCancellationException when deleting comments
Cause: CommentItem's coroutineScope cancelled when removed from UI
Fix: Pass stable coroutineScope from parent (ModalBottomSheet) to CommentItem

B. Comment ID Mapping
Problem: Comments loaded without document ID, couldn't delete
Fix: Changed getComments() to manually map doc.id to Comment.id
  comments = snapshot.documents.mapNotNull { doc ->
      doc.toObject(Comment::class.java)?.copy(id = doc.id)
  }

- Reduced from 300dp to 225dp (25% shorter)
- Input bar properly positioned with navigation bar padding
